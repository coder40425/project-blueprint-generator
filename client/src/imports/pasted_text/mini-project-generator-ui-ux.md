Design a modern, premium, highly responsive web application UI/UX for a SaaS-style AI platform called “Mini Project Generator” by SkillDzire.

The platform helps students, freshers, and developers instantly generate complete software project blueprints using AI.

The user enters a simple project idea prompt such as:

* “Food Delivery App”
* “Student Attendance System”
* “AI Resume Builder”
* “E-commerce Website”

The AI then generates:

1. Project Title
2. Short Description
3. Recommended Tech Stack
4. Core Features / Modules
5. Database Tables / Schema Suggestions
6. API Requirements
7. Development Roadmap / Steps
8. Estimated Timeline
9. Suggested Folder Structure / Architecture
